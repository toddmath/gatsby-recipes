# Setup Gatsby with `gatsby-plugin-typography`

[gatsby-plugin-typography](https://www.gatsbyjs.org/packages/gatsby-plugin-typography/) is a Gatsby plugin for utilizing the [Typography](https://kyleamathews.github.io/typography.js/) library with minimal configuration.

Based on https://www.gatsbyjs.org/tutorial/part-three

---

Install necessary NPM packages

<NPMPackage name='gatsby-plugin-typography' />
<NPMPackage name='react-typography' />
<NPMPackage name='typography' />
<NPMPackage name='typography-theme-fairy-gates' />

---

Install Gatsby Plugin

<GatsbyPlugin
  name='gatsby-plugin-typography'
  options={{ pathToConfigModule: `src/utils/typography` }}
/>
